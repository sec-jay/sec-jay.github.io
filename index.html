论文与投稿Publications / Under Review

Unknown malware detection based on hyperspherical embedding and out-of-distribution sample detection

亮点：以超球面嵌入与置信度校准提升未知恶意软件检测鲁棒性。
Highlight: Robust OOD malware detection with hyperspherical embedding and calibrated confidence.
Under Review · IEEE IoTJ
问题—贡献：针对零日样本与高误报问题，提出超球面嵌入 + OOD 置信度判定的统一检测框架。
Problem–Contribution: A unified hyperspherical embedding + OOD confidence framework for zero-day detection with lower false positives.
  • 问题：已知样本训练的检测器难以发现零日恶意软件且误报率高。
  • 方法：超球面嵌入 + SupCon + vMF 建模 + 马氏距离置信度判定。
  • 特征：融合字符串、API、操作码三类静态特征，筛选 500 个重要特征。
  • 评估：Family-10 已知样本 + OOD 未知样本，对比 KNN、Malconv、CIDER 等基线。
OOD malware detection with hyperspherical embedding, SupCon, vMF modeling, and Mahalanobis confidence. Multi-source static features (string/API/opcode) and benchmarked against KNN, Malconv, and CIDER.

An Evolutionary Strategy-Enhanced Conditional GAN for High-Evasion and Low-Trace Adversarial Malware Generation

亮点：ES-CGAN 在 PE 合规扰动下实现高规避与低痕迹。
Highlight: ES-CGAN achieves high evasion with minimal, PE-compliant perturbations.
Under Review · Venue TBD
问题—贡献：在离散字节域不可微与规避/隐蔽权衡下，提出 ES‑CGAN 双层优化生成框架。
Problem–Contribution: ES‑CGAN bridges non-differentiable byte space and evasion/stealth trade-offs via a two-level optimization framework.
  • 问题：对抗样本在“高规避 vs 低痕迹”之间难以平衡。
  • 方法：ES-CGAN（CMA-ES + CGAN）双层优化，解决离散字节域不可微问题。
  • 扰动：PE 结构合法扰动（DOS/存根、节区空闲填充、新节区注入）。
  • 结果：在 VirusTotal 上峰值规避率 79.43%，平均 ASR 69.60%，PBR 低至 0.9%。
ES-CGAN with CMA-ES + CGAN for discrete PE perturbations. Achieved 79.43% peak evasion on VirusTotal, 69.60% mean ASR, and 0.9% PBR.
